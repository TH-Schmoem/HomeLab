# HomeLab
Personal HomeLab infrastructure using Proxmox, UniFi, OPNsense, Docker, VLAN networking, media automation, Nextcloud and game server hosting.


## Core Components

- Proxmox VE Hypervisor (Virtual Machines & Containers)
- OPNsense Firewall (Network Segmentation, VPN, Security)
- UniFi Switching & Wireless Infrastructure
- VLAN-based Network Architecture
- Docker Application Stack
- Media Automation (Plex, Sonarr, Radarr, Prowlarr, Overseerr)
- Nextcloud Private Cloud
- AMP Game Server Management
- WireGuard Remote Access
- Cloudflare Tunnel External Access
- Centralized Monitoring and Backups

## Goals

- Secure and segmented network architecture
- Enterprise-style infrastructure layout
- High availability and scalability
- Automated updates and maintenance
- Clean documentation and reproducible deployments
- Learning platform for enterprise networking and virtualization concepts

This repository serves as a complete technical reference including architecture diagrams, configuration examples, deployment checklists, network planning spreadsheets and operational documentation.
